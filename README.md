# 売価の自動決定に向けた分析メモ（簡潔版）

本リポジトリでは、売価の自動決定に向けて「需要（price）と供給（cost）のバランスを見ながら粗利を最大化する」ための前段分析を行う。現在は可視化による仮説検証を中心に進めている。

## 目的
- 月別・年別の粗利構造を把握し、売価決定に必要な特徴を抽出する
- 高粗利商品／低粗利商品の寄与と数量の関係を可視化する

## これまでの主な検討内容
- 月別の原価・売価・数量の推移可視化
- JAN別で、全年度を集約した粗利率の分布（Q1/平均/Q3）を月ごとに表示
- 個数で重み付けした月別・粗利の箱ひげ図（案①）
- 月×粗利×個数の散布図（案③）
- 年別で、JAN単位の平均粗利率×平均個数を散布図で可視化
  - 個数分布の広がり対策として `log10` 変換
  - 粗利率=0.5、log10(個数)=1.5で象限分割し、象限別の粗利寄与率を表示

## 次のステップ候補
- 需要（price）と供給（cost）が一致し粗利最大となる売価の推定モデル化
- 年・月・商品特性ごとの売価最適化に向けた特徴量設計と検証
